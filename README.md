# sot-defensive-analysis-cltfc
Video-based defensive analysis project conducted with the Charlotte FC Academy. Includes tagged data from 17 shots-on-target clips, charts, error analysis, and a full presentation for coaching staff. Focus areas: method of attack, defensive density, and mistake-driven chances. PowerPoint and PDF included.
